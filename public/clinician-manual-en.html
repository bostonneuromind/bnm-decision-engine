1. System Overview

1.1 What This System Does

CNA v7 is a clinical decision support system for 5 beta clients (part-time therapy clients).

v7 Core Additions (Round 7-10): - Differential Diagnosis Engine (DDE): Client data → Bayesian differential diagnosis (ADHD/GAD/MDD) - Comorbidity Decomposition: Decompose clinical scores by diagnostic source - Cognitive Simulator: Counterfactual simulation of clinical outcomes when cognitive axes change - Adaptive Re-routing: Auto-reconstruct diagnosis from 4-week outcome

Core Functions:

  • Fischer Dynamic Skill Theory: cognitive 5-axis → clinical 5-axis prediction
  • Auto-analysis of relationships among 5 clinical axes (Attention/Learning/Peak Performance/Anxiety/Depression)
  • DSM-5-TR based differential & comorbidity diagnosis (v7)
  • Auto-generation of training protocols (4-8 weeks) when client selects area
  • Decision-making based on Karl Friston's Active Inference framework
  • System auto-learns from outcomes (Variational Belief Update + Diagnostic Re-routing)

1.2 Target Users

  • Clinicians: BCN, PhD, LMHC, LMFT, mental health professionals
  • Clients: 18+ (beta), Learning + Peak Performance + mild anxiety/depression domains
  • Exclusions: Acute psychosis, active suicidal crisis, uncontrolled bipolar disorder

2. Core Concepts

2.1 Fischer Cognitive 5-Axis (Layer A)

Measurable cognitive abilities:

Axis Measurement Tool Normal Range
Sustained Attention CPT (Continuous Performance Test) Omissions Z = -0.5 to +0.5
Working Memory N-back (2-back, 3-back) Accuracy 70-85%
Emotional Regulation HRV RMSSD + VSR RMSSD 25-50ms
Time Awareness Time estimation task Error 5-15%
Self-Awareness OMR (Observable Model Refinement) Response depth

Fischer Level: L7.0 ~ L13.0 (adult typical range)

2.2 Clinical Outcome 5-Axis (Layer B)

Client-selectable treatment areas:

Area Primary Measurements
Attention CPT, ASRS
Learning N-back, learning retention
Peak Performance Time estimation, HRV
Anxiety HRV, GAD-7
Depression PHQ-9, BA count

2.3 4-Vector Input

  • V1 State: Current measurements + symptom scores
  • V2 Desire: Client's desired outcomes + motivation
  • V3 Skill: Fischer 5-axis Levels
  • V4 Trajectory: Cumulative past session data

2.4 Free Energy 6-Component Decision

Each card scored on 6 components:

  1. Accuracy (Symptom): Client symptoms ↔ card targets matching
  2. Accuracy (Biomarker): Measurements ↔ card expected profile
  3. Complexity Penalty: Fischer level distance (Vygotsky ZPD)
  4. Preference: Match with client's selected area
  5. Epistemic Value: Information gain potential
  6. Pragmatic Value: Evidence + immediate effect

Weights auto-adjust based on client phase: - New client: epistemic priority (data gathering) - Stable client: pragmatic priority (efficacy) - Emergency client (priority tier): emergency phase auto-activates, pragmatic highest priority

4. First Client Workflow

4.1 Session 1 (Initial Assessment) — 75-105 min (v7 adds ~15 min for temporal interview)

Step 1: Client Registration (5 min)

  • Client initials + ID number (e.g., AK001)
  • DOB, gender
  • Chief complaints
  • Medications
  • Guardian consent (if needed)

Step 2: Entry Protocol 5 Measurement (30 min)

Use paper measurement sheets. Five measurements:

  1. Time Estimation (5 min) → time_awareness
  2. HRV (5 min, Polar H10 or Apple Watch) → emotional_regulation
  3. CPT (15 min, paper or digital) → sustained_attention
  4. N-back (10 min, 2-back 3-back) → working_memory
  5. VSR (optional, 5 min) → additional data

Step 3: Symptom Data Collection (15 min)

  • PHQ-9 (depression screening) — includes suicide risk item
  • GAD-7 (anxiety screening)
  • Subjective symptom scores (0.0-1.0):
  • Inattention
  • Distractibility
  • Executive Dysfunction
  • Anxiety Somatic
  • Anxiety Cognitive
  • Low Mood
  • Low Motivation
  • Social Withdrawal

Step 4 (v7 NEW): Temporal History Interview (15 min) ⭐

DDE accuracy's foundation. Detailed guide in Section 6. Core 8 questions:

  1. "Age symptoms first appeared?" → onset_age
  2. "Childhood/adolescence/adult?" → onset_category
  3. "Chronic or episodic?" → course
  4. "How many months ongoing?" → duration_months
  5. "Consistent across settings or context-specific?" → pervasiveness
  6. "Worsens with stress? Improves with rest?" → stress_triggered
  7. "Family history?" → family_history
  8. "Prior treatment response?" → prior_treatment_response

Step 5: Client Goal Elicitation (10 min)

  • Select one area to improve most:
  • Attention
  • Learning Efficiency
  • Peak Performance
  • Anxiety Management
  • Mood Stability
  • Motivation level (0-10)
  • Perceived barriers

5. Entry Protocol 5 Measurement Guide

5.1 Time Estimation

Procedure: - Instruct: "Raise hand when you think 30 seconds have passed" - Repeat 5 times - Calculate error % per trial - Average error % → time_awareness Level

Interpretation: | Avg Error | Fischer Level | Clinical Note | |-----------|---------------|---------------| | 0-5% | L11+ | Highly accurate | | 5-12% | L9-10 | Normal | | 12-25% | L8 | Weak | | 25%+ | L7 or below | Clinical attention |

5.2 HRV Measurement (RMSSD)

Procedure: - Use Polar H10 or Apple Watch - 5-min resting breathing (seated) - Auto-calculate RMSSD - Record LF/HF ratio also

Interpretation: | RMSSD (ms) | Status | |------------|--------| | 50+ | Very good | | 35-50 | Good | | 25-35 | Average | | 15-25 | Weak | | < 15 | Clinical attention |

5.3 CPT

Procedure: - Use Conners CPT, IVA-2, or custom tool - 15-min standard protocol - Results: omissions, commissions, RT variability

Z-score Conversion: - Norm mean = 0, SD = 1 - Client score → Z-score (positive = better) - Omissions Z < -1.5 = clinical concern

5.4 N-back

Procedure: - 2-back 5 min, 3-back 5 min - Record accuracy % - Calculate d-prime

Interpretation: | 2-back Accuracy | Fischer Level | |-----------------|---------------| | 90%+ | L11+ | | 75-90% | L9-10 | | 60-75% | L8 | | < 60% | L7 |

5.5 VSR (Optional)

  • 800-cell stem reading
  • Acoustic analysis → 4D mapping
  • Consistency score 0-1

6. Temporal History Collection Guide (v7 NEW)

6.1 Why It's Decisive

DDE's 4-source likelihood weights temporal at 30%. Missing temporal info drops differential accuracy significantly.

Clinical basis: Differentiating ADHD vs GAD vs MDD hinges on temporal patterns. - ADHD: Childhood onset (< 12 yo), chronic, consistent across settings - GAD: Adult onset possible, 6+ months, stress-triggered - MDD: 2+ week episodes, pervasive environmental impact

6.2 8-Item Interview

1. Onset Age (onset_age)

  • Question: "What age did these symptoms first appear?"
  • Significance: Under 12 = strong ADHD signal
  • Response: Integer (e.g., 8)
  • If unknown: null (OK, supplement with other info)

2. Onset Category (onset_category)

  • Question: "When did it start — childhood, adolescence, or adulthood?"
  • Options: "childhood" | "adolescence" | "adult" | "any_age"
  • Significance: Childhood = strong ADHD marker

3. Course (course)

  • Question: "Has it been chronic, or episodic — coming and going?"
  • Options: "chronic" | "episodic" | "fluctuating"
  • Significance: ADHD chronic, MDD episodic, GAD chronic or episodic

4. Duration in Months (duration_months)

  • Question: "How many months have these symptoms persisted?"
  • Response: Integer months
  • Thresholds:
  • MDD: 0.5+ (2+ weeks)
  • GAD: 6+ (DSM-5-TR criterion)
  • ADHD: 60+ (5+ years chronic)

5. Pervasiveness (pervasiveness)

  • Question: "Consistent across work, home, social — or specific to certain situations?"
  • Options:
  • "cross_setting" — all environments (ADHD signal)
  • "context_dependent" — specific situations only (anxiety signal)
  • "all_setting" — depression presentation
  • Significance: DSM-5-TR core ADHD criterion

6. Stress Triggered (stress_triggered)

  • Question: "Do symptoms worsen with stress and improve with rest?"
  • Response: true | false
  • Significance: true = anxiety/depression, false = ADHD

7. Family History (family_history)

  • Question: "Family members with similar symptoms or psychiatric diagnoses?"
  • Response: String list (e.g., ["father_ADHD", "mother_GAD"])
  • Significance: ~50% genetic (especially strong for ADHD)

8. Prior Treatment Response (prior_treatment_response)

  • Question: "Past medications or therapy? What worked?"
  • Response: dict (e.g., {"SSRI": "no_response", "stimulant": "good_response"})
  • Significance: SSRI non-response + stimulant good response = strong ADHD primary signal (McIntosh 2009)

6.3 Data Entry Example

from cna_core import TemporalHistory

temporal = TemporalHistory(
    onset_age=8,
    onset_category="childhood",
    course="chronic",
    duration_months=240,
    pervasiveness="cross_setting",
    stress_triggered=False,
    family_history=["father_ADHD"],
    prior_treatment_response={"SSRI": "no_response"},
)

6.4 Interview Tips

  • Order: onset age → category → course → duration → pervasiveness → stress → family → treatment
  • Unknown items: null or unknown (don't force) — empty data better than inaccurate
  • Reliability: Family history from client report — supplement with direct family interview if possible
  • MDD suspicion: Ask "Almost every day for past 2 weeks?" (DSM-5-TR criterion)

Primary Thresholds: - Posterior > 40%: strong signal, primary confirmed - Posterior 30-40%: primary but review recommended - Posterior < 30%: primary uncertain (review required)

Secondary: Posterior > 20%

Confidence: - 0.7+: very reliable - 0.5-0.7: moderately reliable - < 0.5: additional evaluation needed

8.2 4-Source Likelihood

Source Weight Description
Symptom match 40% Client complaint vs diagnosis pattern
Cognitive match 20% Cognitive 5-axis weakness pattern
Biomarker match 10% CPT/HRV/N-back etc. objective measures
Temporal match 30% Temporal info (differential's key)

Key insight: Symptom + Temporal = 70%. These two are most decisive.

8.3 Differential Markers Met/Missed

DSM-5-TR criteria auto-checked per diagnosis:

ADHD Inattentive: - ✓ Cross-setting consistency (pervasiveness == "cross_setting") - ✓ Onset before age 12 (onset_age < 12) - ✓ Not better explained by mood/anxiety (symptom pattern analysis)

GAD: - ✓ Excessive worry 6+ months (duration >= 6) - ✓ Worry > attention deficit (anxiety > inattention) - ✓ HRV RMSSD < 25ms

MDD: - ✓ Anhedonia or low mood 2+ weeks - ✓ Rumination (vs ADHD distractibility) - ✓ Cognitive slowing (vs ADHD inconsistency)

8.5 Auto-Flag for Clinician Review

System auto-recommends review when: - Low diagnostic certainty (top posterior < 40%) - Insufficient temporal info (onset/course unknown) - Difficult differential (1st vs 2nd gap < 10%) - ADHD suspicion but onset age unconfirmed

When flagged, clinician makes final decision (not system auto-decide).

11.4 Re-routing Decision Interpretation

【 Diagnostic Re-routing Decision 】
  Primary: adhd_inattentive → gad
  Secondary: [] → [adhd_inattentive]
  Changed: yes
  Change type: primary_swap
  Response pattern: primary_no_response_secondary_improved_anxiety
  Posterior change: 0.35
  ⚠ Clinician review needed

Change Types: - no_change: Diagnosis maintained (response fit 0.60+) - secondary_added: New comorbid diagnosis - primary_swap: Primary diagnosis changed (e.g., ADHD → GAD) - diagnosis_dropped: Previous diagnosis excluded

Response Patterns: - expected_response_strong: Primary target responds well (>+0.15) - partial_response: Partial response - primary_no_response_secondary_improved_*: Primary unresponsive + only other axis improved → diagnosis doubt - treatment_no_response: No response at all - diagnosis_revision_from_response: Response matches different diagnosis pattern

11.5 Clinical Re-routing Scenarios

Scenario A: ADHD primary → stimulant card → no effect, anxiety worsens - System catches: "Expected ADHD primary but stimulant non-response + anxiety increase" - Re-routing: Change primary to GAD, ADHD secondary - Clinical meaning: GAD caused ADHD-like symptoms (Eysenck ACT)

Scenario B: MDD primary → BA card → no mood change, only HRV improves - System catches: "MDD treatment with no depression response" - Re-routing: GAD secondary likelihood ↑ - Clinical meaning: Chronic anxiety more primary than depression

Scenario C: ADHD primary → stimulant card → attention ↑, learning ↑ - System catches: "Response as expected (response fit 0.85+)" - Re-routing: No change, diagnosis confidence ↑

11.6 Safety

  • Auto-rollback if volatility > 0.15
  • Protected cells immune to learning (e.g., emotional_regulation→anxiety)
  • Auto-pause learning if mean outcome < -0.30
  • Re-routing only activates when posterior change > 0.20
  • Primary swap auto-flags clinician review
